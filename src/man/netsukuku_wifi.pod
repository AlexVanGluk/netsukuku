=head1 NAME

netsukuku_wifi - configures the specified wifi card to join Netsukuku

=head1 SYNOPSIS

netsukuku_wifi I<wifi_interface>

=head1 DESCRIPTION

B<Netsukuku_wifi> is a bash script, which takes as its first argument the name
of the wifi network interface to be used in order to join in Netsukuku. Then
it sets the wifi interface in Ad-Hoc mode and its essid to "netsukuku".
At this point the B<netsukuku_d> daemon is ready to be launched,

=head1 EXAMPLES

netsukuku_wifi wlan0
